- Images
- Videos
- Documents
- and more...

![menu](https://i.imgur.com/GVZvzNf.png)

## Upload
It works in a very simple way, by clicking in the Upload button
you will select the file you want to upload:
![upload](https://i.imgur.com/hznK5zQ.png)

after this, the file will be uploaded and loaded into the dashboard:
![grid](https://i.imgur.com/dYixFxu.png)

you can also download the file or preview it:
![preview](https://i.imgur.com/WU8XRIY.png)

## Dashboard Organization
You can also organize the dashboard by grid or list files
![grid](https://i.imgur.com/dYixFxu.png)
![list](https://i.imgur.com/KEJFBuY.png)
